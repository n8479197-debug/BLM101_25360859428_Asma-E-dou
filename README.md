# BLM101_25360859428_Asma-E-dou
Öğrenci Bilgileri: Asma Eıdou, 25360859428
Proje Konusu: 4. Grup: Makine Dili ve Brookshear Mimarisi
Proje Açıklaması:
 
Kod Ne Yapıyor?
Bu proje, Brookshear makinesi için yazılmış basit bir HEX komut çözücüsüdür.  
Kullanıcıdan 4 haneli bir HEX kodu alır ve bu kodun hangi işlemi temsil ettiğini
metin olarak açıklar.

Nasıl Çalıştırılır?
- Kod Python programlama dili ile yazılmıştır.
- Program çalıştırıldığında kullanıcıdan 4 haneli bir HEX kodu girmesi istenir.
- Girilen kod analiz edilir ve ekrana açıklaması yazdırılır.

Kurulum Gerekli mi?
- Bilgisayarda Python kurulu olmalıdır.
- Ekstra bir kütüphane veya kurulum gerekmemektedir.

 Kodun Çalışma Mantığı:
1. Program, kullanıcıdan 4 haneli bir HEX kodu alır.
2. Girilen kodun uzunluğu kontrol edilir.
3. Kodun sadece 0-9 ve A-F karakterlerinden oluşup oluşmadığı kontrol edilir.
4. Kod; opcode, register ve adres olacak şekilde parçalara ayrılır.
5. YouTube Linki : https://youtu.be/KvWzxLK_4y4
6. Opcode değerine göre hangi işlemin yapılacağı belirlenir.
7. İlgili işlem ekrana açıklama olarak yazdırılır.
8. Eğer opcode desteklenmiyorsa kullanıcı bilgilendirilir.
